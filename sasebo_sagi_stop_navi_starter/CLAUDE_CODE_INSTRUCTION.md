# Claude Codeへの実装指示

あなたは、佐世保市民向けの詐欺被害防止静的サイト「させぼ詐欺ストップナビ」を実装してください。

## 最優先目的

怪しい電話・SMS・メール・LINE・SNSなどを受けた人が、送金、個人情報入力、URLタップ、アプリ導入、折り返し電話をする前に一度止まり、安全な相談先へ進めること。

## 使用技術

- HTML5
- CSS3
- Vanilla JavaScript
- JSON
- 外部フレームワーク不要
- ビルド不要
- index.htmlを直接開いても基本表示できる構成
- 将来GitHub PagesやXサーバーへ配置できる構成

## 実装条件

1. モバイルファースト
2. 360px幅から崩れない
3. 本文18px以上
4. 主要ボタン20px以上
5. タップ領域44px以上
6. 下部に「相談する」固定ボタン
7. 緊急表示は110へワンタップ発信
8. 電話番号はtelリンク
9. 外部リンクは新しいタブで開き、外部サイトであることを表示
10. 個人情報を送信・保存しない
11. localStorageにも診断内容を保存しない
12. アクセス解析コードを入れない
13. 「詐欺ではありません」「安全です」と判定しない
14. 不明な場合は相談へ誘導
15. 診断結果に免責を表示
16. 公的機関のロゴ・キャラクター・画像を無断使用しない
17. JSONの文言はHTMLエスケープして表示
18. URLパラメータから任意HTMLを挿入できないようにする
19. キーボード操作、フォーカス表示、aria-labelを実装
20. prefers-reduced-motionに対応

## 簡易確認機能

質問は最大4段階。

1. 連絡手段
2. 相手が求めるもの
3. よくある言葉
4. すでに行った行動

判定は安全側に倒す。

- 現金・カードを取りに来る、脅迫、ニセ警察、遠隔操作、既に送金 → 緊急または危険度高
- お金、暗証番号、カード、アプリ、電子マネー、秘密、SNS投資 → 危険度高
- URL付き不在通知など → 注意
- 該当なし → 公式窓口で確認

結果画面には次を必ず表示。

- 結論
- 今すぐ中止する行動
- 次に行うこと
- 相談先
- 家族への相談文をコピーするボタン
- 最初からやり直すボタン

## トップページ

最初の画面に次を表示。

「その電話、そのメッセージ。
急いで答えなくて大丈夫です。」

「お金を払う前に。URLを押す前に。名前や口座番号を伝える前に。」

主要ボタン：
- 電話
- SMS
- メール
- LINE・SNS
- 訪問・郵便
- すでに支払った・入力した

## アプリページ

Android、iPhone、分からない、の3選択を最初に表示。
警察庁推奨アプリ2種類を公平に掲載し、特定事業者を過度に推さない。
「無料」「警察庁推奨」「外部ストアへ移動」を明記。
Androidだけ国際電話番号の一括ブロック機能があることを明記。
公式情報へのリンクを必ず付ける。

## 被害後ページ

以下で分岐。

- URLを押しただけ
- ID・パスワードを入力
- カード情報を入力
- 遠隔操作アプリを導入
- 振込・送金
- 現金・カードを渡した

注意：
遠隔操作アプリの削除だけを先に促さない。まず通信遮断、金融機関・警察への相談、記録保存を案内する。

## デザイン

行政サイトより簡単で、子どもっぽくしない。
白、紺、淡い青を中心。
危険表示だけ赤。
イラストは自作の単純な線画またはCSSアイコン。外部画像は不要。
カード型UI、十分な余白、短い文章。
高齢者が焦っていても読めることを優先。

## 完了時に行うこと

- 全ページのリンク切れ確認

- スマホ360px、390px、430px、PCで確認
- 電話リンク確認
- コピー機能確認
- JavaScript無効時にも相談先だけは表示
- README.mdに更新方法を書く
- 変更したファイル一覧を報告する
