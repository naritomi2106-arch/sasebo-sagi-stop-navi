# サイト仕様書

## 1. サイト名

させぼ詐欺ストップナビ

サブタイトル：
電話・SMS・メール・LINE・SNS
「これ、詐欺かも」と思ったときに

## 2. 目的

詐欺について勉強するサイトではなく、怪しい連絡を受けた人が送金・入力・折り返し・アプリ導入などを行う前に、一度止まり、安全な相談先へ進める行動支援サイトとする。

## 3. 主な対象者

- 佐世保市内の高齢者
- 高齢者の家族
- ケアマネジャー
- 地域包括支援センター職員
- 民生委員・児童委員
- 訪問介護・通所介護などの支援者
- 一般市民
- SNS型投資詐欺やロマンス詐欺の被害が心配な現役世代

## 4. 最重要原則

- 最初に「止める」
- 結論を先に表示する
- 一画面の情報量を抑える
- 「安全」と判定しない
- 緊急時は110へ
- 被害後は警察・金融機関・カード会社への行動を優先
- 外部リンク先は公式サイトだけに限定
- 個人情報を入力させない
- 利用履歴や診断内容をサーバーへ送信しない

## 5. トップページ構成

### ファーストビュー

見出し：
その電話、そのメッセージ。
急いで答えなくて大丈夫です。

本文：
お金を払う前に。
URLを押す前に。
名前や口座番号を伝える前に。
届いた連絡の種類を選んでください。

固定緊急表示：
犯人が家に来る、現金やカードを取りに来る、脅されている場合は110番してください。

### 主要選択ボタン

- 電話がかかってきた
- SMSが届いた
- メールが届いた
- LINE・SNSで連絡が来た
- 訪問・郵便・ハガキ
- すでに支払った・入力した

### 補助導線

- 家族・支援者の方へ
- 詐欺対策アプリ
- 佐世保の相談先
- よくある手口を見る

## 6. 簡易確認フロー

### ステップ1
連絡手段を選択する。

### ステップ2
相手が求めていることを選択する。

- お金・振込
- 現金・カードを渡す
- 口座番号・暗証番号
- カード番号
- URLを押す
- アプリを入れる
- ビデオ通話
- 電子マネーを買う
- 誰にも話さない
- よく分からない

### ステップ3
状況を選択する。

- まだ何もしていない
- URLを押した
- 個人情報を入力した
- ID・パスワードを入力した
- カード情報を入力した
- アプリを入れた
- 振り込んだ・送金した
- 現金やカードを渡した

### ステップ4
結果を表示する。

結果は3段階とする。

- 緊急：110番
- 危険度が高い：連絡を中止し、警察・消費生活センターへ相談
- 注意が必要：相手が示した連絡先を使わず、公式窓口へ確認

「問題ありません」「詐欺ではありません」という表示は禁止。

## 7. 結果画面

必ず次を順番に表示する。

1. 結論
2. 今すぐやめること
3. 次にすること
4. 相談先
5. 家族への相談文コピー
6. 最初から確認し直す

例：
詐欺の可能性が高い連絡です。
これ以上、返信・送金・入力をしないでください。

## 9. デザイン

- スマホ幅360pxから崩れない
- 本文18px以上
- 主要ボタン20px以上
- タップ領域は最低44px
- 危険表示だけ赤
- 通常は白・紺・淡い青を中心
- アイコンと文字を併用
- 1画面の選択肢は最大6個程度
- 下部に「相談する」固定ボタン
- ふりがな対応を将来追加しやすい構造
- WCAG 2.1 AAを意識
- キーボード操作可能
- aria-labelを設定
- 色だけで危険度を表現しない

## 10. 免責表示

本サイトは、入力内容から詐欺かどうかを確定するものではありません。
少しでも不安がある場合は、相手との連絡を止め、警察や消費生活センターなどの公式窓口へ相談してください。

## 11. 運用

- 手口データはJSONで管理
- 相談先情報もJSONで管理
- 最終確認日をページ下部に表示
- 3か月ごとに電話番号、受付時間、外部リンクを確認
- 新しい手口はJSON追加で反映
